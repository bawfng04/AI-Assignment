\begin{abstract}
Minesweeper is a classic logic puzzle game that challenges players to clear a grid of hidden mines without detonating any of them. This report presents the design and implementation of a complete Minesweeper game in Python using the Tkinter graphical user interface library. Furthermore, to fulfill the requirements of the Artificial Intelligence assignment, an intelligent agent (AI Solver) was developed and integrated into the game. The core gameplay heavily utilizes the Iterative Depth-First Search (DFS) algorithm with an explicit stack data structure to handle the ``Flood Fill'' mechanic robustly, bypassing Python's strict recursion limits. The integrated AI Solver employs a sophisticated 3-tier reasoning architecture, transitioning from basic Propositional Logic constraints to advanced DFS Backtracking for Constraint Satisfaction, and smoothly falling back to Educated Probalistic Guesses when absolute certainty is unattainable. Empirical tests demonstrate that the AI agent achieves a win rate of approximately 96\% on standard small grids ($5\times5$) and 95\% on medium grids ($9\times9$). The project successfully demonstrates the practical application of fundamental data structures, graphical interface design patterns (MVC), and classic AI deduction mechanisms in a single, cohesive software product.

\textbf{Từ khóa (Keywords):} Minesweeper, Artificial Intelligence, Depth-First Search (DFS), Propositional Logic, Constraint Satisfaction, Backtracking.
\end{abstract}
